The embodiment of the invention discloses a multiECU wiring harness connection method and a wire harness control box The method and the wire harness control box are used for connecting multiple ECUsto achieve multiwireharness normalization and improve the portability of multiECU connection The method includes the following steps that an MCU is used for detecting whether a wire harness is connected with an electronic control unit ECU or not if yes the MCU detects the type of the ECU and receives the ECU types input through an ECU selection button according to the ECU types the MCUcontrols a drive circuit and a switch array so that a corresponding communication signal and a switching signal are in a conducting state
