 CNB The invention relates to a switching observer controlbased photovoltaic inverter system control method which specifically comprises the following steps in step S a photovoltaic multimode switching inverter system is built in step S a fuzzy model of a photovoltaic inverter system is built based on physics principles and an expression method of a TS model in step S a switching observation controller is designed based on the fuzzy model of the photovoltaic inverter system the photovoltaic inverter system is enabled to run stably and therefore steadystate performance thereof can beensured According to the method disclosed in the invention the switching observation controller is designed to solve a problem of multimode switching of the photovoltaic inverter system the photovoltaic inverter system is enabled to run stably and therefore the steadystate performance thereof can be ensured
