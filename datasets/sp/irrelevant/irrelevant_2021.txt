 CNA     The present invention provides a control method device and system for an active suspension control mode wherein the method includes acquiring the current active suspension control mode of the vehicle and the target working condition information of the target road section The historical active suspension control mode matched with the operating condition information and the historical vehicle attitude level corresponding to each historical active suspension control mode the first candidate mode and the second candidate mode are determined according to the above information and when the vehicle is on the target road section Or when entering the target road section the active suspension control mode of the vehicle is adjusted according to at least one of the first candidate mode and the second candidate mode and the current mode adjustment strategy of the vehicle The scheme of the present invention judges the rationality of the active suspension control mode and actively advises and reminds the active suspension control mode so that the user can switch to the active suspension control mode suitable for working conditions in time which is beneficial to ensure driving control Sex and ride comfort
