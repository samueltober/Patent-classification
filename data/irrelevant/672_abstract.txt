The invention provides a novel dimmable intelligent LED bulb lamp The novel dimmable intelligent LED bulb lamp comprises an external information acquisition unit an information processing unit a multifunctional auxiliary unit and a safety alarm unit The external information acquisition unit is used for sensing illumination intensity information and human body information of the environment the information processing unit is used for controlling the illumination mode of the LED bulb lamp according to a user instruction and detected external information and comprises a user interaction module an information fusion decision module a data storage module and a light adjustment control module the multifunctional auxiliary unit is used for providing more diversified control modes for a user and comprises an audio control module a wireless transmission module and a priority setting module the safety alarm unit is used for giving an alarm when a user calls for help and the power supply is abnormal According to the invention the intelligent degree is high the safety is good diversified control modes can be provided for the user and the user experience is improved
