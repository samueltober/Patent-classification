 CNA The invention relates to an integrated multiinterface debugger based on USB conversion and belongs to the technical field of debuggers The debugger comprises a hardware circuit unit The hardware circuit unit comprises a USB input module a power supply module a charging management module a USB HUB chip a USB to serial port chip a fourway serial port to level chip a USB to internet accesschip an internet access transformer a DB male interface a DB female interface a consol interface an RS interface an RJ interface a Type C charging port a Micro USB charging port and aLightning charging port The debugger meets the equipment debugging requirements of most hardware and debugging can be implemented on a computer with a USB port In addition an integrated cable is integrated a connecting cable does not need to be carried the size of the debugger is greatly reduced meanwhile convenience is improved and work efficiency and user experience are improved
