The invention relates to a servo control system and method for a multiviewfield thermal imager The servo control system comprises a power supply filtering module a power supply conversion modulean asynchronous communication module a main control module a main control reset module a motor driving module a highprecision sensor position signal modulation and demodulation module a signal conditioning module a reference voltage generation module a connector and a limit switch input module The servo control system realizes precise position servo control for four motors by adopting a main controlling with the performance meeting requirements to serve as a core control device a motor driver with ultrasmall volume and a highprecision position signal acquisition module
