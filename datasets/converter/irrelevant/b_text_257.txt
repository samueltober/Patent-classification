 CNB The invention discloses a universal ground detection board for satellite payload based on a PXI bus The detection board comprises an FPGA integrated PXI master board an OC command module an LVDS communication module an RS differential communication module an AD acquisition module and a temperature telemetry module The universal ground detection board provided by the invention can realizethe generalization of the onboard payload ground detection board and save the ground detection development cost required for different payloads and by integrating the functions of an OC command LVDS communication RS AD acquisition and temperature telemetry the universal ground detection board simplifies the cable connection improves the integration and reduces the PXI chassis resourcesoccupied by the ground detection board
