 CNA The invention provides a multichannel radio frequency power testing device The device comprises a power sensor a lownoise amplification circuit module and a highspeed DAQ data acquisition modulethe number of the power sensors is at least two The input ends of all the power sensors are connected with a tested sample the output ends of all the power sensors are connected with the input endof the lownoise amplification circuit module the output end of the lownoise amplification circuit module is connected with the input end of the highspeed DAQ data acquisition module the power sensors receivethe test signal of the tested sample and converts the test signal into an analog signal the analog signal is amplified by the lownoise amplification circuit module to complete a signal conditioning function the signal conditioning function is output to the highspeed DAQ data acquisition module for highspeed analogtodigital conversion and finally the signal power is converted into a digital power value The beneficial effects of the multichannel radio frequency power testing deviceare that the device is accurate in measurement achieves the multiport parallel testing andis high in testing speed
