 CNU The utility model relates to a multichannel bus instrument The multichannel bus instrument comprises a microcontroller the N signal measurement processing modules comprise M digital signal processing modules and NM analog signal processing modules the digital signal processing modules are connected with the microcontroller and the analog signal processing modules are connected with the microcontroller through an analogtodigital converter the power supply control module is connected with the microcontroller and supplies power to the N measurement processing modules and the bus protocol stack processing module is connected with the microcontroller and is connected with external equipment through the first communication interface and the bus protocol stack processing module comprises a plurality of Ethernet bus protocols The multichannel bus instrument provided by the utility model can update a bus protocol on line which is equivalent to replacement of a bus module so that remote operation can be realized
