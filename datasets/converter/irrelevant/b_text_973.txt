 CNU     The utility model relates to a portable visualized ship equipment structure vibration tester A power management module a sensor module a signal conditioning module an AD conversion module an FPGA module a liquid crystal display module and an externally expanded data memory are integrated into a whole to form a portable Vibration tester the power management module includes a reference voltage supply unit and a charging unit the sensor module converts the vibration signal into a voltage signal and the voltage signal is input to the signal conditioning module the signal conditioning module amplifies the weak signal input from the sensor module for AD conversion provides signals the AD conversion module converts the analog signal input by the signal conditioning module into a digital signal for the FPGA to process data the FPGA module is used for vibration signal processing and transmits the processed data to the LCD module and external expansion Data storage The utility model can realize the rapid test and analysis of the structural vibration of the ship equipment
