Systems and methods for power management are disclosed herein In one disclosed embodiment the battery charging system uses a closedloop charging servo target to simultaneously charge the battery andor to system load using a closedloop charging servo target based on measurements taken by one or more gauges To provide power and a battery charger In some embodiments multiple power sources may be utilized simultaneously according to priority levels determined for one or more of the multiple power sources coupled to the battery as well as for example a charging profile specifying one or more battery charging parameters  In some embodiments the priority level of a given power source is not fixed Rather the priority level for a given power source can vary based on the characteristics of a given power source In some embodiments priority levels for multiple power sources are implemented using cascaded voltage target values
