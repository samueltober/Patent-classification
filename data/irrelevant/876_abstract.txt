The utility model discloses a grassroots meteorological disaster early warning and issuance system which includes a weather monitoring module a mountain environment monitoring module a river environment monitoring module a front alarm module a power supply module a data collector a first wireless transceiver module and a second wireless Transceiver module decoding unit data processing unit storage module and background alarm module the collection interface of the data collector is connected to the output interface of the weather monitoring module mountain environment monitoring module and river environment monitoring module and the data collector passes the data signal The first wireless transceiver module sends the data signal to the server the server receives the data signal through the second wireless transceiver module the decoding unit converts the data signal and the data processing unit analyzes and processes the converted data signal The utility model can effectively warn the grassroots meteorological disasters through the analysis and processing of the collected parameters so as to broadcast notifications to the grassroots people in advance and reduce losses
