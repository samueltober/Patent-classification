 CNB     A power adapter used to supply power to mobile devices The power adapter includes a processor and an interface for data communication and power transmission with the mobile device a memory inside the housing of the power adapter and an ACDC power conversion circuit electrically coupled to the processor  The ACDC power conversion circuit is configured to receive AC power input and convert the AC power input to DC power output for the mobile device via the interface The processor is configured to identify a load associated with the mobile device connected to the DC power output set the DC power output based on the load receive a backup from the mobile device via the interface Data and storing the backup data from the mobile device in the memory
