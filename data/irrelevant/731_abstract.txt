The invention provides an LED display unit and an LED display device and the LED display unit comprises an input interface which is used for receiving an input DC voltage and a video signal the timesequence controller is used for respectively sending a scanning signal and a gray scale signal generated by a video signal received by the input interface to the row driver and the column driver so as to drive the LED display module to display a preset picture and the output interface is respectively connected with the input interface and the time schedule controller and is used for transmittingthe input direct current voltage transmitted by the input interface and the video signal transmitted by the time schedule controller According to the LED display unit the LED display equipment adopting the structure tends to be lighter and thinner
