The invention provides a TVS test system based on TCPIP communication The TVS test system comprises an upper computer a lower computer a power supply and a TVS leakage current test module The upper computer is connected with a power supply the power supply is in communication connection with the lower computer through the TVS leakage current test module The TVS leakage current test module comprises a current acquisition circuit and a TVS leakage current test position the power supply is electrically connected with the current acquisition circuit through the TVS leakage current test position the current acquisition circuit is electrically connected with the lower computer and the TVS leakage current test position is used for installing a TVS The upper computer the lower computer the power supply and the TVS leakage current test module are adopted to form the TVS tube performance test system the performance of the TVS tube is detected by utilizing the communication of the upper computer and the lower computer the automatic scanning test is realized and the test efficiency is improved
