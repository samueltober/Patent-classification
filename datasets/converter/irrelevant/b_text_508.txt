 CNA The invention discloses a TypeC display and a TypeC cascade display system The TypeC display comprises a display main body a first TypeC interface a second TypeC interface a signal input interface and a controller wherein the first TypeC interface the second TypeC interface and the signal input interface are arranged on the display main body and the controller is arranged in the display main body the first TypeC interface the second TypeC interface and the signal input interface are all connected with the controller the first TypeC interface is used for inputting power supply voltage and receiving audio and video signals the signal input interface is used for receiving audio and video signals of terminal equipment and the second TypeC interface is used for outputtingthe audio and video signals of the terminal equipment and outputting the power supply voltage According to the technical scheme the convenience of the cascade display is improved while the cost ofthe cascade display is reduced
