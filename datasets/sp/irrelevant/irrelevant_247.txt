 CNB The invention provides an automobile active suspension and electric power steering system integrated control device and a working method The device comprises a data acquisition module an AD conversion module an electronic control unit and an actuator The actuator comprises a stepping motor and a booster motor The data acquisition module is sequentially connected with the AD conversion module and the electronic control unit and the electronic control unit is then communicatively connected to the stepping motor and the booster motor An electric power steering system is combined with anactive suspension system for integrated control so that the mutual interference between the two systems can be better solved At the same time the system configuration is streamlined information sharing is achieved between the systems and limited sensor and controller resources are fully utilized to improve system reliability and reduce system cost
