The embodiment of the present invention discloses a signal transmission network a chip and a signal processing device The signal transmission network includes an impedance unit including a signal input terminal and a plurality of signal output terminals The impedance unit is used to receive the signal input from the signal input terminal The input signal is converted into multiple different current signals after gain adjustment to different degrees and multiple different current signals are respectively provided to multiple signal output terminals the gate switch unit includes multiple switch input terminals and switch output terminals The switch input terminal is connected to a plurality of signal output terminals and the gate switch unit is used to selectively turn on one switch input terminal and one switch output terminal to output a corresponding current signal The signal transmission network structure of the embodiment of the present invention is simple and can further increase the bandwidth of the signal transmission network
